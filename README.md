# Nova Studio

Nova Studio is a modern digital agency website built using Next.js, React, Neon PostgreSQL, and Vercel. The project showcases services, portfolio projects, company statistics, and a contact form with data stored in a PostgreSQL database.

## Features

* Responsive modern design
* Services section
* Portfolio section with project images
* Statistics section
* Contact form connected to database
* Admin login page
* Admin dashboard for managing content
* Deployed on Vercel
* Database hosted on Neon PostgreSQL

## Tech Stack

* Next.js
* React
* JavaScript
* Neon PostgreSQL
* Vercel

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Move into the project folder:

```bash
cd nova-studio
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file and add:

```env
DATABASE_URL=your_neon_database_url
ADMIN_PASSWORD=your_admin_password
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
